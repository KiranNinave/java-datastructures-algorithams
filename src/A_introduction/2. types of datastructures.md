types of datastructures

1. primitive DS
    this datastructures are provided by programming language
    - integer
    - character
    - float
    - boolean

2. non primitive DS
    A. physical DS
        this datastructures physicaly store in RAM
        - array
        - linklist

    B. logical DS
        the datastructures needs physical datastructures for there implementaions
        - stack
        - queue
        - tree
        - graph
        - hashtable
